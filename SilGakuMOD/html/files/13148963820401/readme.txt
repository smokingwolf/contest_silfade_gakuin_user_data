【本ファイルについて】
この度はバトルシュミレータをDL頂き真に有難うございます。

本ファイルはショートストーリーより戦闘の実験を行う為のプログラムです
DataOfficial内のShortStoryフォルダ内に入れてお使いください

【使用方法】
○最初に使用するキャラクタを決定します。
自キャラ1人目から敵キャラ3人目までを決定してください。

＊主人公は1人目の時のみ選択可能です
＊2人目で「なし」を選ぶと自動で3人目も「なし」になります。

○次にパラメータ設定を行います。
選択キャラのパラメータを変更します。
主人公は主人公設定でも変更可能です。

＊戦闘時は、成長度によってもパラメータが変わります。
影響をなくす際は成長度を100に設定してください。

○成長度設定
キャラクタの成長度を設定します。
初年度、2年度、3年度より選択の上で、希望する月を選択してください。
直接、「何ヶ月目」かを指定する事もできます。

＊成長度は30日毎の成長の為、全月30日で計算しています。
よって時間が経つことに実際の成長度とはずれが発生する事になりますのでご注意ください

○主人公設定（主人公選択時）
主人公のスキル、装備、ポテンシャル等を設定します。

○ルール設定
ルール設定を行います
一部ルールは設定後解除ができなくなりますので注意してください。
